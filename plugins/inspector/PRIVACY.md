# Privacy Policy — data-anim Inspector

**Last updated:** 2026-03-20

## Data Collection

The data-anim Inspector Chrome extension does **not** collect, store, or transmit any user data. All functionality runs entirely within your browser.

## Permissions

- **activeTab**: Used to inject the inspector UI into the current tab when you click the extension icon.
- **scripting**: Used to inject the inspector content script into the current tab.

These permissions are only used when you explicitly activate the extension. No data is accessed, collected, or sent to any server.

## Third-Party Services

This extension does not use any third-party analytics, tracking, or data collection services.

If you have any questions, please open an issue at https://github.com/ryo-manba/data-anim/issues.
