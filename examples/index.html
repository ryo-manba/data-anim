What is data-anim?

A declarative animation library. No JavaScript authoring required — just HTML attributes.

index.html 
<!-- Just add attributes -->
<h1
  data-anim="fadeInUp"
  data-anim-trigger="scroll">
  Hello World
</h1>

<div
  data-anim="zoomIn"
  data-anim-trigger="hover"
  data-anim-once>
  Hover me!
</div>

<ul
  data-anim-stagger="100">
  <li data-anim="fadeInRight">Item 1</li>
  <li data-anim="fadeInRight">Item 2</li>
  <li data-anim="fadeInRight">Item 3</li>
</ul>

Hello World

This element faded in from below when it scrolled into view.

Hover me!

This element zooms in when you hover over it.

Staggered Items

These cards appear one after another with a stagger delay.

Why data-anim?

Six reasons to choose a declarative approach over imperative animation code.

🛡️

Anti-FOUC

3-layer flash prevention ensures elements stay hidden until their animation fires. No flicker, ever.

⚡

< 2KB gzip

Zero dependencies. Ships as a single file — smaller than most loading spinners.

🎯

4 Triggers

Scroll, load, click, and hover. Combine them to build complex interactions without writing code.

🎬

30+ Animations

Fade, slide, zoom, bounce, rotate, and attention seekers. All built in, all customizable.

📱

Responsive

Disable or swap animations per breakpoint with data-anim-mobile and data-anim-tablet attributes.

♿

Accessible

Automatically respects prefers-reduced-motion. Your users' preferences are honoured by default.

How It Works

Three steps. No build tools. No configuration files.

1

Add Script

Include the library via CDN or npm. One line in your HTML head.

<script src="data-anim.js"></script>
2

Write Attributes

Add data-anim attributes to any HTML element you want to animate.

<div data-anim="fadeInUp">
3

Done ✨

No build step. No config. Your elements animate on scroll, load, click, or hover.

Quick Start

Copy this complete HTML file and open it in your browser. That's it.

quickstart.html 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport"
    content="width=device-width, initial-scale=1.0">
  <title>My Page</title>
  <script src="https://unpkg.com/data-anim"></script>
</head>
<body>

  <h1 data-anim="fadeInUp">
    Hello data-anim!
  </h1>

  <p data-anim="fadeInUp"
     data-anim-delay="200">
    I animate on scroll with a 200ms delay.
  </p>

  <div data-anim="zoomIn"
       data-anim-trigger="hover">
    Hover to zoom!
  </div>

  <ul data-anim-stagger="100">
    <li data-anim="fadeInRight">One</li>
    <li data-anim="fadeInRight">Two</li>
    <li data-anim="fadeInRight">Three</li>
  </ul>

</body>
</html>
