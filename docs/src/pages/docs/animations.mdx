# Animations

data-anim ships with 30+ animations out of the box. Each one is a pure CSS keyframe animation, optimized for performance.

## Fade

Smooth opacity transitions, optionally combined with directional movement.

## Using Animations

Every animation name is passed to the `data-anim` attribute:

<CodeBlock
  code={`<div data-anim="fadeInUp" data-anim-duration="800ms">
  This element fades in from below over 800ms
</div>`}
  lang="html"
/>

See [Attributes](/docs/attributes) for all customization options, or try them in the [Playground](/playground).
