# Getting Started

## Installation

### CDN

<CodeBlock
  code={`<script src="https://unpkg.com/data-anim@latest/dist/data-anim.js"></script>`}
  lang="html"
/>

### npm

<CodeBlock code={`npm install data-anim`} lang="bash" />

<CodeBlock code={`import 'data-anim';`} lang="js" />

## Basic Usage

<CodeBlock
  code={'<div data-anim="fadeInUp">\n  Hello, animated world!\n</div>'}
  lang="html"
/>

<AnimationDemo name="fadeInUp" />

See [Animations](/data-anim/docs/animations) for all 30+ built-in animations.

## Triggers

By default, animations play on scroll. See [Triggers](/data-anim/docs/triggers) for all options.

<CodeBlock
  code={'<!-- Hover -->\n<div data-anim="pulse" data-anim-trigger="hover">\n  Hover me\n</div>\n\n<!-- Click -->\n<button data-anim="bounce" data-anim-trigger="click">\n  Click me\n</button>'}
  lang="html"
/>

## Customization

<CodeBlock
  code={'<div\n  data-anim="slideInLeft"\n  data-anim-duration="800"\n  data-anim-delay="200"\n  data-anim-easing="ease-out"\n  data-anim-once\n>\n  Customized animation\n</div>'}
  lang="html"
/>

See [Attributes](/data-anim/docs/attributes) for the full reference and [Stagger](/data-anim/docs/stagger) for cascading sequences.
