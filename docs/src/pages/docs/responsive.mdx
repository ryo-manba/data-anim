# Responsive

Not every animation works well on mobile. data-anim lets you swap or disable animations based on screen size.

<ResponsiveDemo />

## data-anim-mobile

Swap to a simpler animation on mobile (< 768px). Desktop gets the original, mobile gets the override.

<CodeBlock
  code={'<div\n  data-anim="slideInLeft"\n  data-anim-mobile="fadeIn"\n>\n  Slides on desktop, fades on mobile\n</div>'}
  lang="html"
/>

## data-anim-disable

Disable the animation entirely on a specific breakpoint.

<CodeBlock
  code={'<!-- No animation on mobile -->\n<div data-anim="slideInLeft" data-anim-disable="mobile">\n  Desktop only\n</div>\n\n<!-- No animation on desktop -->\n<div data-anim="fadeInUp" data-anim-disable="desktop">\n  Mobile only\n</div>'}
  lang="html"
/>

| Value | Effect |
|-------|--------|
| `"mobile"` | Disabled on < 768px |
| `"desktop"` | Disabled on ≥ 768px |

## Reduced Motion

`prefers-reduced-motion: reduce` automatically disables all animations. No manual handling needed.
